# OpenSCAD Models

A collection of parametric 3D models written in [OpenSCAD](https://openscad.org/).

## Models

### hollow_cylinder_cutouts.scad

A vertical hollow cylinder with two circular cutouts:

- **Large cutout** — faces the +X direction, centered vertically
- **Small cutout** — faces the +Y direction (90° around from the large one), centered vertically

All dimensions are parameterized at the top of the file for easy adjustment.

## Usage

Open any `.scad` file in OpenSCAD, tweak the parameters, and render (F6) or preview (F5).
